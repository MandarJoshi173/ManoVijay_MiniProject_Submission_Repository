# Mental Health Therapy Management System

## Description

This project is a Django-based web application developed to provide mental wellness support by connecting users with therapists and enabling appointment management, therapist blogs, notifications, and productivity tools.

---

## Features

- User Authentication
- Therapist Dashboard
- Appointment Booking
- Therapist Blogs
- Notification System
- Pomodoro Timer
- Profile Management
- Feedback Module

---

## Technologies Used

### Frontend
- HTML
- CSS
- JavaScript
- Bootstrap

### Backend
- Python
- Django

### Database
- SQLite

---

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
```

Navigate into project folder:

```bash
cd djangoProject
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python manage.py runserver
```
